What is this?

This is a GitHub Action and dynamic site that generates YouTube cards for a given channel in your GitHub README.

How do I use it?

Check out the GitHub Readme for instructions on how to set up the action and use the cards in your repo or profile page.

How do I customize it?

Check out the Advanced Configuration section of the GitHub Readme for a list of input options you can use in addition to the channel_id. You can also check out the Wiki for more information on how to use specific features.
